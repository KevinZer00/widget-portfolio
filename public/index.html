Project 1 ×

Dungeon Crawler Game

Technologies: Phaser.JS, CSS, HTML, Tiled (Tilemap Editor)

Live Demo

This is a single-level dungeon crawler 2D game.

This project was originally meant to be my gamified portfolio. The objective of the game is to control a knight and find all four treasure chests. Each treasure chest opens up a modal explaining a particular project I completed.

The game features a variety of monsters with differing healths and movement speeds. The game incorporates key game components such as collisions, enemy-player detection, combat system, health system, and menu UI.

Project 2 ×

Genesis AI

Technologies: Vanilla JS, CSS, HTML

Live Demo

This is a mock site for a fictional AI named Genesis built with vanilla JS.

Meant as a satirical site aimed towards the emerging landscape of AIs, this site revolves around a new fictional AI that promises to boost user productivity and provide companionship.

The site features an Ubuntu inspired theme, a player-computer Tic-Tac-Toe game simulation with the min-max algorithm implementation, a Spotify embedded music player, and a chat box.

NOTE: There is a hidden feature within the site. Please continue chatting with the AI despite incoherent responses.

Project 3 ×

Widget-Based Portfolio

Technologies: Node.JS, Vanilla JS, CSS, HTML

Live Demo

This project is the portfolio you are currently looking at!

The portfolio was inspired by those retro dashboards you would typically see in movies and games.

This project features a weather API widget with integrated geolocation, a web audio API music player with equalizer, a terminal that accepts user commands, current time/date, hardware stats retrieval and dynamic CPU utilization graph (through the use of os-utils library)
