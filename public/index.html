Hi there! My name is Kevin and welcome to my portfolio!

I am an aspiring front-end web developer. All my knowledge was obtained through online sources such as The Odin Project, YouTube, and W3Schools!

Making fun, engaging, and creative projects is what I enjoy doing! I discovered that web development is a perfect outlet for me to do so.

I hope you enjoy the rest of the portfolio! Please play around with all the little widgets (especially the terminal).

Project 1 ×

Dungeon Crawler Game

Technologies: Phaser.JS, CSS, HTML, Tiled (Tilemap Editor)

Live Demo

This is a single-level dungeon crawler 2D game.

This project was originally meant to be my gamified portfolio. The objective of the game is to control a knight and find all four treasure chests. Each treasure chest opens up a modal explaining a particular project I completed.

The game features a variety of monsters with differing healths and movement speeds. The game incorporates key game components such as collisions, enemy-player detection, combat system, health system, and menu UI.

Project 2 ×

Genesis AI

Technologies: Vanilla JS, CSS, HTML

Live Demo

This is a mock site for a fictional AI named Genesis built with vanilla JS.

Meant as a satirical site aimed towards the emerging landscape of AIs, this site revolves around a new fictional AI that promises to boost user productivity and provide companionship.

The site features an Ubuntu inspired theme, a player-computer Tic-Tac-Toe game simulation with the min-max algorithm implementation, a Spotify embedded music player, and a chat box.

NOTE: There is a hidden feature within the site. Please continue chatting with the AI despite incoherent responses.

Project 3 ×

Widget-Based Portfolio

Technologies: Node.JS, Vanilla JS, CSS, HTML

Live Demo

This project is the portfolio you are currently looking at!

The portfolio was inspired by those retro dashboards you would typically see in movies and games.

This project features a weather API widget with integrated geolocation, a web audio API music player with equalizer, a terminal that accepts user commands, current time/date, hardware stats retrieval and dynamic CPU utilization graph (through the use of os-utils library)

Project 5 ×

Invictus Coaching

Technologies: GSAP, Vanilla JS, CSS, HTML

Live Demo

This project is a mock site for a client's personal training business.

The site features various creative CSS styling such as parallax scrolling (achieved through image layers) and GSAP text/image loading animations. The overall aesthetic, as discussed with the client, is an aggressive bodybuilding tone, with a black and red color palette.

Project 6 ×

3D Planet Size Comparison

Technologies: Three.JS, GSAP, CSS, HTML

Live Demo

This project is a fun demo of the Three.JS framework combined with GSAP scroll triggers.

The project utilizes the Three.JS framework to introduce 3D models. The planet models are simple three-dimensional spheres with texture skins wrapped around. There are GSAP scrolling triggers that will scroll through each planet, from smallest to largest.

Project 7 ×

Horror Game

Technologies: Vanilla JS, CSS, HTML

Live Demo

This project is a single level horror game utilizing JavaScript's canvas.

As a big fan of horror games such as Outlast and Phasmophobia, I wanted to create a small project that could potentially evoke a similar feeling of horror.

This project features dual layered Canvas elements - one for the background, and one for the VHS and flashlight effect. All image assets were AI generated via Midjourney.
